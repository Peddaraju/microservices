# Spring Cloud Gateway
* Simple, yet effective way to route to API's
* Provide cross cutting concerns
  * Security
  * Monitoring/Metrics
* Build on top of Spring WebFlux (Reactive Approach)
* Features
  * Match routes on any request attribution
  * Define Predicates and Filters
  * Integrate with Spring cloud Discovery client(Load Balancing)
  * Path Rewriting
